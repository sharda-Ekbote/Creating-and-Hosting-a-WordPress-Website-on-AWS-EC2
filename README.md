Creating and Hosting a WordPress Website on AWS EC2

An end-to-end guide on deploying a functional WordPress site using a LAMP stack (Linux, Apache, MySQL, PHP) on an Amazon EC2 instance.

Project Overview

This project demonstrates how to set up a scalable and reliable web hosting environment on AWS. It covers instance provisioning,
security group configuration, and the manual installation of the WordPress core and its dependencies.

🛠 Tech Stack

Cloud: Amazon Web Services (EC2)

OS: Ubuntu Linux

Web Server: Apache

Database: MySQL

Language: PHP

CMS: WordPress
